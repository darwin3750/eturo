# eTuro

## What?
eTuro is an open source learning platform that uses a Massive Open Online Course (MOOC) Model to deliver educational materials online. eTuro aims to equip educators with essential skills and knowledge to teach in an era where digital learning is ubiquitous and often mandatory in todayâ€™s situation. Powered by the community, eTuro reaps the collective knowledge and experience of professionals, students and enthusiasts to share their perspective and knowledge amongst the community.

## Why?
To upgrade online learning. So no one gets left behind.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
